# heroes-adventures

Simple react-based web game. Currently in beta it is 100% client-side project.

## Installation
Clone repository and execute:

```
yarn
yarn start
```
or if you're using npm:
```
npm install
npm start
```

## Folder structure
```
├───components
├───helpers
├───json
├───services    
└───styles
```
